Text formatting

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and headers elements are block elements. An inline element keeps its content on the same line as the content before and after it. It dose NOT create a new line. The bold, italics and underline elements are inline elements.

Images

An is a block element which inserts pictures into the document. An image element is a self-closing element which means it dose not have an end tag. It has one attribute, src, which is a filepath that points to the images location. Those filepaths can be either absolute or relative.

Images Attributes

There are 4 attributes commonly used for images. width and height are used to resize the images. alt allows to use text-description of image for accessibility and broken images. title allows a hover-over text to appear.

Width and Height

The width and height can be measured in many different ways, but the more common ones are pixels and percentages. Using pixels, you set the specific, precise size based on pixels. using percentages, you set the dimensions based on a percentage of the entire page. Percentage based re-sizing will often respect aspect ratio while using pixels may not.

Alt Attribute

EVERY picture should at LEAST have the alt attribute. The alt attribute helps with screen readers for visually impaired users and in the cases that the image is broken
